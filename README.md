# conky-toggle

하모니카 4.0의 바탕화면 모니터링 기능을 온/오프 하는 프로그램

 * HamoniKR (>= 4.0) 지원
 * dependancy : conky, conky-all, conky-manager

# Install

## HamoniKR 사용자의 경우
터미널을 열고(Ctrl+Alt+T) 아래 명령어를 입력하세요.

```
sudo apt update
sudo apt install -y conky-toggle
```

# Usage
 * 프로그램 > `바탕화면 모니터링 온/오프` 실행

프로그램을 실행하면 바탕화면의 모니터링 프로그램이 켜지거나 꺼집니다.
